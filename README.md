# The Truth

AI is the most capable reasoner humanity has ever built — and we waste it on flattery. Ask a frontier model a hard question and you get hedged both-sides-ism, or whatever it thinks you want to hear. If AI really is the smartest intelligence on the planet, it should be able to *resolve* debates, not launder them.

This repo is an attempt to put scaffolding around AI that forces it past sycophancy and toward the capital-T Truth behind any topic — public policy, ethics, metaphysics, history, science. Every real debate has a correct side. The goal here is to find it.
